Remarks

Calculates cosine similarity between two vectors using formula. 

    A · B
—————————————,
||A|| * ||B||
where A and B are vectors, i is the index of the vector element; · is the dot product; ||A|| is length of A, calculated as √(∑(A²i)).

LINQ version should be optimized (using ZLinq/SIMD) prior production usage, current implementation is only for reference.

Exceptions
Type Condition
ArgumentException

if vector sizes are not equal OR one of vectors has Euclidean length of 0.
